Features

  • Requires explicit action - Cannot be dismissed by clicking overlay (unlike Dialog)
  • Async operation support - Handles async callbacks with loading states
  • Confirm/Cancel callbacks - Separate handlers for both actions
  • Body scroll lock - Automatically locks body scroll when open
  • Accessible - Uses alertdialog role and proper ARIA attributes
  • Error handling - Gracefully handles errors in callbacks, keeps dialog open on failure
