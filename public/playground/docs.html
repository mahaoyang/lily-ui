Getting started

Lily UI is a lightweight component library built with Alpine.js and Tailwind CSS, inspired by Radix Themes.

1Install dependencies

First, make sure you have Alpine.js and Tailwind CSS installed in your project.

npm install alpinejs
npm install -D tailwindcss

2Include the stylesheet

Add the Lily UI stylesheet to your HTML head.

<link rel="stylesheet" href="/dist/output.css">

3Initialize Alpine.js

Import and start Alpine.js in your page.

<script type="module">
  import Alpine from 'alpinejs';
  window.Alpine = Alpine;
  Alpine.start();
</script>

4Add theme configuration

Wrap your app with theme attributes on the body tag.

<body
  data-scaling="100%"
  data-radius="medium"
  class="rt-Theme radix-themes"
  data-accent-color="iris">
  <!-- Your content -->
</body>

Styling

Lily UI uses CSS custom properties and Tailwind CSS for flexible styling.

CSS Variables

The theme uses Radix Colors for consistent color scales. Each color has 12 shades:

  • --color-1 to --color-3: Backgrounds
  • --color-4 to --color-6: Interactive backgrounds
  • --color-7 to --color-8: Borders
  • --color-9 to --color-10: Solid colors
  • --color-11 to --color-12: Text

Tailwind Integration

All Radix color scales are available as Tailwind classes:

<!-- Background -->
<div class="bg-sand-2">...</div>

<!-- Text -->
<span class="text-accent-11">...</span>

<!-- Border -->
<div class="border border-sand-6">...</div>

Customizing radius

Set the data-radius attribute on body to change corner radius:

<body data-radius="none">   <!-- Sharp corners -->
<body data-radius="small">  <!-- Subtle rounding -->
<body data-radius="medium"> <!-- Default -->
<body data-radius="large">  <!-- More rounded -->
<body data-radius="full">   <!-- Fully rounded -->

Color

Lily UI uses Radix Colors for a comprehensive and accessible color system.

Accent colors

Set the accent color using data-accent-color attribute:

<body data-accent-color="iris">   <!-- Purple (default) -->
<body data-accent-color="blue">   <!-- Blue -->
<body data-accent-color="violet"> <!-- Violet -->
<body data-accent-color="teal">   <!-- Teal -->
<body data-accent-color="jade">   <!-- Green -->
<body data-accent-color="red">    <!-- Red -->

Color scales

Use accent colors with the accent- prefix in Tailwind classes:

<button class="bg-accent-9 text-white">Primary</button>
<button class="bg-accent-3 text-accent-11">Secondary</button>

Dark mode

Enable dark mode by adding the dark class to your body element.

Basic usage

Toggle dark mode by adding or removing the dark class:

<body class="dark">
  <!-- Dark mode enabled -->
</body>

With Alpine.js

Use Alpine.js to manage theme state:

<body
  x-data="{ theme: 'light' }"
  :class="theme === 'dark' ? 'dark' : ''">

  <button @click="theme = theme === 'light' ? 'dark' : 'light'">
    Toggle theme
  </button>
</body>

Persist preference

Save theme preference to localStorage:

<body
  x-data="{ theme: localStorage.getItem('theme') || 'light' }"
  x-init="$watch('theme', val => localStorage.setItem('theme', val))"
  :class="theme === 'dark' ? 'dark' : ''">
  ...
</body>

View Transitions

Add smooth transitions when switching themes:

function toggleTheme() {
  if (document.startViewTransition) {
    document.startViewTransition(() => {
      theme = theme === 'light' ? 'dark' : 'light';
    });
  } else {
    theme = theme === 'light' ? 'dark' : 'light';
  }
}

Typography

Typography utilities for consistent text styling.

Font sizes

Use Tailwind's text size utilities with our color system:

<h1 class="text-4xl font-bold text-sand-12">Heading 1</h1>
<h2 class="text-2xl font-semibold text-sand-12">Heading 2</h2>
<p class="text-base text-sand-11">Body text</p>
<small class="text-sm text-sand-10">Small text</small>

Text colors

  • text-sand-12 - High contrast text (headings)
  • text-sand-11 - Body text
  • text-sand-10 - Muted text
  • text-accent-11 - Accent colored text

Font weight

Recommended font weights:

  • font-bold (700) - Headings
  • font-semibold (600) - Subheadings
  • font-medium (500) - Labels, buttons
  • font-normal (400) - Body text
