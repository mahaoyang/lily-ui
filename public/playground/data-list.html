Usage Guide

HTML Structure

<dl class="data-list-root data-list-horizontal data-list-size-2">
  <div class="data-list-item">
    <dt class="data-list-label">Label</dt>
    <dd class="data-list-value">Value</dd>
  </div>
  <!-- More items... -->
</dl>

Available Classes

  • .data-list-root - Container element (dl)
  • .data-list-item - Wrapper for each term-value pair (div)
  • .data-list-label - Term/label element (dt)
  • .data-list-value - Value/description element (dd)

Size Variants

  • .data-list-size-1 - Small size
  • .data-list-size-2 - Medium size (default)
  • .data-list-size-3 - Large size

Orientation

  • .data-list-horizontal - Labels and values side by side (grid layout)
  • .data-list-vertical - Labels and values stacked (flex column)

Item Alignment (Horizontal only)

  • .data-list-align-baseline - Baseline alignment (default)
  • .data-list-align-start - Start alignment
  • .data-list-align-center - Center alignment
  • .data-list-align-end - End alignment
  • .data-list-align-stretch - Stretch alignment

Label Color

  • data-accent-color - Add this attribute to labels to use accent color
  • .data-list-high-contrast - High contrast label text
